博客采用极简终端风格设计，暗色背景 + 等宽字体，无多余装饰。这本身就是一种审美声明。

### 自我定位
- Bio: "嵌入式开发者，关注自动化控制与机器视觉。用 C 和 Python 构建东西。"
- 技能标签：嵌入式开发、OpenMV/OpenCV、PCB设计、C/C++/Python、PID控制、硬件电路设计、3D打印、MATLAB/Simulink、Keil/CubeMX/VSCode、SolidWorks

### 项目清单（12个公开仓库）
1. **markdown_everything** — 20+ 文档格式转 Markdown 的自动化工具（PowerShell）
2. **voice-claude** — VoxCPM2 TTS 语音合成/声音克隆工具链（Python）
3. **AGV_car** — CDIO 自动导引小车控制系统（C）
4. **Balance-the-car** — STM32 平衡小车 PID 控制（C）
5. **detect_my** — 目标检测实验（Python）
6. **mediapipe_RSP_game** — MediaPipe 手势识别猜拳游戏（Python）
7. **C-Python-NumPy-PyTorch** — C 开发者定制的深度学习入门教程
8. **thesis-skill-hub** — 硕士论文审查与写作 Skill 聚合
9. **easy_mbti** — MBTI 人格分析工具（Python）
10. **deepseek-monitor** — DeepSeek API 用量监控桌面应用（Python）
11. **llm-pid-tuner** — 基于 LLM 的 PID 自动调参系统
12. **Python-dl-cli** — Python 机器学习学习软件（JavaScript CLI）

### 关键观察
- 项目跨度大：从嵌入式硬件 → 计算机视觉 → AI 工具链
- 每个项目解决一个具体问题，不做「大而全」的框架
- llm-pid-tuner 是典型跨界项目：用 AI 优化传统控制算法
- C-Python-NumPy-PyTorch 体现了「用自己会的东西学新东西」的方法论
