# Excel 主表列映射与更新规范

## 工作表结构

主表文件名：`2027届自动化专业保研夏令营信息汇总.xlsx`
工作表名：`2027自动化夏令营总表`

### 行布局
- 行 1：表头（标题）
- 行 2：口径说明行（每次更新时刷新，含核验日期、院校数、重点变化摘要）
- 行 3：列标题行
- 行 4 起：数据行（每行一所院校）

### 列映射

| 列字母 | 列号 | 字段 | 填充规范 |
|--------|------|------|----------|
| A | 1 | 层次 | C9 / 985 / 211 / 双一流 |
| B | 2 | 学校 | 院校全称 |
| C | 3 | 学院 | 目标学院全称 |

## 截止时间格式（E 列）

### 单学院
```
2026-07-01 17:00
```

### 多学院（用 `；` 分隔，每个学院带 `~` 前缀）
```
人工智能研究院~2026-07-01 17:00；力学与工程科学学院~已截止(6/22)；智能学院~已截止(6/25)
```

### 聚合平台换算（带"约"字）
```
约2026-06-29
约2026-06-29（6/23新发通知）
```

### 已截止
```
2026-06-25（已截止）
已截止(6/25)
```

### 待发布
```
待发布
自动化控制~待发布（校级研学框架6/26发布）
```

## 状态判定规则（F 列）

| 状态值 | 判定条件 |
|--------|----------|
| `已开通（{学院}）` | 查到 2026 年（面向 2027 届）正式通知，且截止时间未过 |
| `已截止（{学院}{日期}截止）` | 截止时间已过，或通知明确"入营名单已公布" |
| `6/27核验仍未见2027届通知` | 核验当天学院官网无 2027 届相关通知 |
| `校级入口已开，{学院}待发布` | 校级入口开放但目标学院方案未发 |
| `已发线上宣讲通知，夏令营待发` | 仅发宣讲通知（如华科 AIA） |
| `每日跟踪：盯{学院}` | 校级汇总持续更新，目标学院待出现 |

## 可报名标记（J 列）

- 可报名：`可报名：{学院}{截止时间}截止`
- 紧急（3 天内截止）：`⚠️可报名：{学院}{截止时间}截止（{N}天，紧急）`
- 已截止：`已截止`
- 无变化确认：保持原值或更新为最新描述
- 待发布：`None`（脚本中设为 None 表示清空）

## 备注字段（G 列）要点

每次更新 G 列应包含：
1. 通知发布日期与截止时间（精确到时分）。
2. 活动时间（如"活动7/13-15"）。
3. 申请条件（如"CET-6 480+"、"需1位意向导师推荐"）。
4. 命名差异（如"非传统夏令营命名"、"导师推荐制"）。
5. 多学院情况（如"自动化考生可关注AI研究院"）。
6. 与自动化方向的相关性说明。

## 技术注意事项

### Python 环境
- 使用 managed Python 3.13.12：`C:\Users\coolkey\.workbuddy\binaries\python\versions\3.13.12\python.exe`
- 运行前设 `PYTHONUTF8=1` 避免中文编码错误。
- openpyxl 安装到项目 `.pylibs` 目录，运行时 `PYTHONPATH` 引用。

### openpyxl 注意事项
- `load_workbook(DST, data_only=False)` 保留公式。
- 保存时若文件被 Excel 打开会 `PermissionError` → 先关闭 Excel 或另存为 `_v2.xlsx`。
- `Alignment(vertical=...)` 只接受 `center/top/bottom`，不接受 `middle`。
- 写入 `None` 会清空单元格；不写入则保留原值。

### 行号映射
- Excel 行号 = 0-based 索引 + 1。
- 数据从行 4 开始：行 4 = 第 1 所院校，行 5 = 第 2 所，依此类推。
- 更新脚本中 `updates[row]` 的 row 直接用 Excel 行号。

### updates 字典结构
```python
updates = {
    行号: {
        5: '截止时间',
        6: '状态',
        7: '备注',
        8: '通知链接',
        9: '通知描述',
        10: '可报名标记',  # None 表示清空
        11: '核验日期',
    },
    # ...
}
```
